# chat-de-terminal
Chat de terminal para conversar enquanto coda.

Requisitos para usar:
Ter o .net instalado(trabalharei para que futuramente não seja necessário);
Entender minimamente de c#;

Comandos Base:
Rodar código: dornet run (servidor);
Criar chat: "1" (Input inicial no lado do cliente);
Entrar em chat: "2" (Input inicial no lado do cliente);
Limpar conversa: //clear (cliente);
Sair da conversa: //quit (cliente).

Assim que todos os integrantes saírwem da conversa, o servidor é fechado.
